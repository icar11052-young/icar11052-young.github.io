윈도우 11 부팅 시 넘버락(Num Lock) 자동 활성화 완벽 해결법

📅 2026-06-30 🏷️ Windows, Keyboard, Troubleshooting

윈도우 11 부팅 시 넘버락(Num Lock) 자동 활성화 완벽 해결법

컴퓨터를 켜고 로그인 화면에서 숫자 PIN(비밀번호)을 치려고 할 때마다 넘버락이 꺼져 있어서 키가 입력되지 않는 불편함을 겪는 경우가 많다. 이 글에서는 레지스트리 수정과 윈도우의 '상태 기억' 동작 방식을 활용해 넘버락을 항상 켜진 상태로 고정하는 방법을 소개한다.

윈도우 넘버락이 꺼지는 원인

윈도우는 로그인 화면 전의 시스템 프로필(.DEFAULT)과 로그인 후 사용자 프로필(HKEY_CURRENT_USER)의 키보드 설정을 독립적으로 관리한다. 또한, 컴퓨터를 종료할 때 마지막 넘버락의 상태(On/Off)를 기억하여 프로필에 자동으로 덮어쓴다. 따라서 넘버락이 꺼진 상태로 시스템을 종료하면 다음 부팅 때 다시 꺼진 채로 켜지게 된다.

해결 방법

1단계: 레지스트리 값 수정

  1. Win + R을 누르고 regedit을 입력한 뒤 엔터를 누른다.
  2. 아래 두 경로로 각각 이동하여 InitialKeyboardIndicators 항목의 값을 수정한다.
  3. 로그인 화면(부팅 직후) 설정 경로:
    HKEY_USERS\.DEFAULT\Control Panel\Keyboard
    ➔ 값 데이터를 2 (또는 2147483650)로 설정
  4. 사용자 계정 로그인 이후 설정 경로:
    HKEY_CURRENT_USER\Control Panel\Keyboard
    ➔ 값 데이터를 2로 설정

2단계: 윈도우에 '켜짐' 상태 기억시키기 (핵심)

레지스트리를 수정했어도 넘버락을 꺼둔 상태에서 재부팅을 하면 윈도우 종료 시 다시 0(꺼짐)으로 설정이 덮어씌워진다.
1. 키보드에서 Num Lock 키를 직접 눌러 불이 들어오게(On) 만든다.
2. 이 상태 그대로 컴퓨터를 다시 시작(또는 종료)한다.
3. 윈도우가 '넘버락이 켜진 상태'를 정상적으로 저장하고 종료하므로, 다음 부팅 때부터 레지스트리 2 값이 유지되며 항상 자동으로 켜진다.
