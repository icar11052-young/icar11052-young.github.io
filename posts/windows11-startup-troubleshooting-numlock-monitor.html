윈도우 11 부팅 문제 해결: 넘버락(Num Lock) 자동 활성화 및 외부 모니터 부팅 멈춤 해결 가이드

📅 2026-06-30 🏷️ Windows, Troubleshooting, Keyboard, Monitor, PowerManagement

윈도우 11 부팅 문제 해결: 넘버락(Num Lock) 자동 활성화 및 외부 모니터 부팅 멈춤 해결 가이드

노트북이나 PC를 사용하면서 부팅할 때마다 사소하지만 매번 사람을 귀찮게 하거나 당황스럽게 만드는 대표적인 부팅 트러블슈팅 두 가지가 있다.

하나는 부팅할 때 숫자 패드의 넘버락(Num Lock)이 자동으로 켜지지 않고 꺼지는 현상이고, 다른 하나는 외부 모니터를 연결한 상태로 부팅하면 화면이 안 뜨고 멈추는(프리징) 현상이다.

이 두 가지 문제는 모두 윈도우 11의 전원 관리 방식 및 드라이버 로드 시점과 밀접한 연관이 있다. 직접 겪고 완벽히 해결한 해결 가이드를 정리하여 공유한다.

🔢 1. 부팅 및 로그인 화면에서 넘버락(Num Lock) 항상 켜두기

컴퓨터를 켜고 로그인 화면에서 숫자 PIN(비밀번호)을 치려고 할 때마다 넘버락이 꺼져 있어서 키가 안 눌리는 불편함을 겪은 적이 있을 것이다. 레지스트리를 수정하고 윈도우의 '상태 기억' 동작을 이해하면 이를 영구적으로 해결할 수 있다.

원인

윈도우는 로그인 화면 전의 시스템 프로필(.DEFAULT)과 로그인 후 사용자 프로필(HKEY_CURRENT_USER)의 키보드 설정을 각각 따로 관리한다. 또한, 컴퓨터를 종료할 때 마지막 넘버락의 상태(On/Off)를 기억하여 프로필에 자동으로 덮어쓴다. 따라서 넘버락이 꺼진 상태로 종료하면 다음 부팅 때 다시 꺼지게 된다.

해결 순서

① 레지스트리 값 수정

  1. Win + R을 눌러 실행 창에 regedit을 입력하고 엔터를 누른다.
  2. 아래 두 가지 경로로 각각 이동하여 InitialKeyboardIndicators 항목의 값을 수정한다.
  3. 로그인 화면(부팅 직후) 설정 경로:
    HKEY_USERS\.DEFAULT\Control Panel\Keyboard
    ➔ 값 데이터를 2 (또는 기존 비트 플래그를 유지하려면 2147483650)로 설정
  4. 사용자 계정 로그인 이후 설정 경로:
    HKEY_CURRENT_USER\Control Panel\Keyboard
    ➔ 값 데이터를 2로 설정

② 윈도우에 '켜짐' 상태 강제 기억시키기 (중요)

레지스트리를 수정했더라도 넘버락이 꺼진 채로 재부팅을 해버리면 윈도우가 종료 단계에서 해당 레지스트리를 다시 0(꺼짐)으로 덮어써 버린다.
1. 컴퓨터 바탕화면에서 키보드의 Num Lock 키를 직접 눌러 불이 들어오게(On) 만든다.
2. 이 상태 그대로 컴퓨터를 다시 시작(또는 종료)한다.
3. 윈도우가 '넘버락이 켜진 상태'를 정상적으로 저장하며 종료되므로, 다음 부팅 때부터 레지스트리 2 값이 유지되며 항상 켜진 채로 부팅된다.

🖥️ 2. 외부 모니터 연결 시 부팅 멈춤(블랙스크린/프리징) 현상 해결

노트북에 USB-C(DP Alt Mode, Power Delivery 포함) 또는 HDMI/DisplayPort로 외부 모니터를 연결해 둔 상태에서 노트북을 켜면, 로고 화면에서 넘어가지 않거나 화면이 아예 검은색으로 유지되며 부팅이 안 되는 치명적인 오류다.

모니터 케이블을 뽑고 켜면 부팅이 잘 되지만, 켤 때마다 케이블을 뽑았다 꽂는 것은 대단히 번거롭다.

원인

윈도우의 '빠른 시작 켜기(Fast Startup)' 기능과의 하드웨어 드라이버 충돌이다. 빠른 시작 기능은 컴퓨터를 완전히 종료하지 않고 시스템 메모리 상태를 하드디스크에 저장한 뒤(하이브리드 최대 절전 모드), 다시 켤 때 이 상태를 그대로 불러와 부팅 속도를 단축한다.

하지만 외부 모니터가 연결되어 전력을 공급받거나 DP 신호를 주고받는 상태에서 커널을 복구하려고 하면, 그래픽 카드 드라이버나 USB-C 포트 컨트롤러가 정상적으로 초기화되지 못하고 시스템이 정지(Crash)해 버리는 것이다.

해결 방법: '빠른 시작 켜기' 해제

이 기능은 SSD를 사용하는 최신 컴퓨터 환경에서는 부팅 속도에 거의 차이를 주지 않으므로 꺼두는 것이 훨씬 안정적이다.

  1. 윈도우 검색창에 제어판을 입력하고 실행한다.
  2. 하드웨어 및 소리 ➔ 전원 옵션으로 들어간다.
  3. 왼쪽 메뉴에서 "전원 단추 작동 설정"을 클릭한다.
  4. 상단의 "현재 사용할 수 없는 설정 변경"을 클릭한다. (관리자 권한 필요)
  5. 아래쪽 '종료 설정' 항목에서 "빠른 시작 켜기 (권장)"의 체크를 해제한다.
  6. 변경 내용 저장을 누르고 컴퓨터를 재부팅한다.

이제 외부 모니터가 몇 대가 연결되어 있든 상관없이 정상적으로 드라이버가 깨끗하게 로드되어 부팅 실패 없이 매끄럽게 켜진다.

💡 요약

  • 넘버락 자동 켜기: 레지스트리 수정 후 꼭 넘버락을 켠 채로 다시 시작을 눌러 상태를 기록해 준다.
  • 모니터 부팅 프리징: 전원 관리 옵션에서 빠른 시작 켜기를 비활성화하여 드라이버 충돌을 방지한다.
