Cursor에서 링크가 크롬으로 열릴 때 — 네이버 웨일 기본 브라우저 설정

Google Antigravity에서 Cursor로 작업 환경을 옮기면서, 평소 쓰던 브라우저는 네이버 웨일인데 AI가 알려준 링크를 누르면 크롬이 열리는 일이 생길 수 있다. 이 글은 윈도우 11에서 웨일을 기본 브라우저로 고정하는 방법을 정리한다.

왜 크롬이 열리나?

Cursor 채팅에 있는 https:// 링크는 대부분 윈도우에 등록된 기본 브라우저로 열린다. 예전에 크롬을 기본으로 써 두었거나, http / https 연결 프로그램이 크롬에 묶여 있으면 웨일을 주로 써도 링크만 크롬으로 갈 수 있다.

참고: Cursor 에이전트 전용 브라우저 자동화는 별도 크롬 엔진을 쓴다. 내가 직접 클릭해서 여는 전자소송·경찰 민원 사이트는 아래 설정으로 웨일에 맞추면 된다.

1단계: 윈도우 11에서 웨일을 기본 앱으로

  1. Win + I로 설정을 연다.
  2. 앱 → 기본 앱으로 이동한다.
  3. 검색창에 웨일 또는 Whale을 입력한다.
  4. 네이버 웨일을 선택한 뒤 기본값으로 설정 버튼을 누른다.

그래도 크롬이면 — 항목별로 수동 변경

「기본값으로 설정」만으로 안 바뀌는 경우가 있다. 같은 화면에서 아래 항목을 하나씩 눌러 네이버 웨일로 바꿔준다.

  • .htm
  • .html
  • HTTP
  • HTTPS

2단계: 웨일 앱 안에서도 한 번 더

  1. 네이버 웨일 실행
  2. 우측 상단 ☰ → 설정
  3. 기본 브라우저 메뉴
  4. 기본 브라우저로 설정 클릭

3단계: Cursor에서 확인

  1. Cursor 채팅에 있는 아무 웹 링크를 클릭한다.
  2. 웨일로 열리면 성공이다.

여전히 크롬이면 Cursor 설정 (Ctrl + ,)에서 browser로 검색해, 외부 링크를 여는 방식 관련 옵션이 있는지 확인한다.

전자소송(ecfs) 쓸 때 참고

대법원 전자소송(ecfs.scourt.go.kr)은 공식 안내상 Chrome·Edge 권장이다. 웨일은 크로미움 기반이라 대부분 동작하지만, 공동인증서·보안 모듈은 웨일에서 로그인·서명이 되는지 한 번 테스트하는 것이 좋다. 안 되면 전자소송만 크롬/엣지를 쓰고 일상 브라우징은 웨일로 나눠 써도 된다.

정리

할 일 위치
기본 브라우저 일괄 변경 설정 → 앱 → 기본 앱 → 웨일 → 기본값으로 설정
http/https 개별 변경 같은 화면에서 HTTP·HTTPS 항목
웨일 내부 확인 웨일 설정 → 기본 브라우저

Antigravity에서 Cursor로 넘어온 첫날, 브라우저만 맞춰도 전자소송·고소 접수 같은 본인이 직접 누르는 행정 작업이 훨씬 편해진다.
