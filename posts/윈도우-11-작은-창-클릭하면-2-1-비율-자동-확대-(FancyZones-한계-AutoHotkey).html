윈도우 11 — 작은 창 클릭하면 2:1 비율 자동 확대 (FancyZones 한계 + AutoHotkey)

📅 2026-07-03 🏷️ Windows, AutoHotkey, FancyZones, Productivity

윈도우 11 — 작은 창 클릭하면 2:1 비율 자동 확대

Cursor 에디터와 Agents 창, 또는 코드와 문서를 2:1로 나란히 두고 작업할 때, 작은 쪽을 클릭하면 그 창이 2/3로 커지고 옆 창이 1/3로 줄어들길 원했다. PowerToys FancyZones의 Win+Ctrl+Alt+1/2 단축키 토글은 동작은 하지만 매번 키를 눌러야 해서 불편했다.

이 글은 FancyZones가 왜 클릭 반응형 리사이즈를 못 하는지, 그리고 AutoHotkey v2로 어떻게 해결했는지 정리한다.

FancyZones로는 왜 안 되나?

FancyZones는 미리 정한 존(zone)에 창을 스냅하는 정적 도구다.

원하는 것 FancyZones
2:1 / 1:2 고정 배치 ✅ 커스텀 레이아웃(66:33 등)
포커스(클릭)한 창이 자동으로 커짐 ❌ 없음

komorebi·GlazeWM 같은 타일링 WM은 키보드 promote로 master 창을 바꿀 수 있지만, 마우스 클릭 한 번으로 2:1을 스왑하려면 결국 AutoHotkey 같은 스크립트가 필요하다.

해결 — AutoHotkey v2 클릭 포커싱

설치 (1회)

  1. AutoHotkey v2 설치
    winget install AutoHotkey.AutoHotkey
  2. 스크립트 실행: window_focus_swap.ahk (Obsidian Vault scratch/window-focus-swap/ 폴더)

사용법

  1. 창 2개를 같은 모니터에 나란히 둔다. (처음 배치는 FancyZones 2:1/1:2 레이아웃이나 아래 단축키)
  2. 작은 쪽 창을 클릭 → 그 창이 2/3, 옆 창이 1/3.
  3. 다시 작아진 쪽을 클릭하면 1:2로 반대 스왑.

보조 단축키

단축키 동작
Win+Alt+Shift+1 지금 두 창 2:1 배치
Win+Alt+Shift+2 지금 두 창 1:2 배치
Ctrl+Alt+F12 스크립트 켜기/끄기

FancyZones의 Win+Ctrl+Alt+1/2는 이후엔 안 써도 된다. 처음 배치만 FancyZones로 해도 되고, 위 단축키만 써도 된다.

FancyZones와 같이 쓸 때

  • 처음 배치만 FancyZones(2:1 / 1:2 레이아웃) → 이후 전환은 AHK 클릭이 담당.
  • 충돌하면 FancyZones 「Windows Snap 재정의」 끄기.

부팅 시 자동 실행

Startup 폴더에 바로가기를 넣으면 로그인마다 실행된다.
UAC 없이 돌리려면 install_autostart_admin.bat을 한 번 실행해 작업 스케줄러에 등록한다. (경로에 공백·한글이 있으면 bat 인용 부호 주의)

문제 해결 (내 환경)

증상 원인 해결
클릭해도 안 바뀜 창 3개 이상 겹침, 높이 너무 작음 같은 모니터에 창 2개만 나란히
PowerShell / Windows Terminal만 안 됨 터미널이 관리자 권한 AHK도 관리자로 재시작 (run_as_admin.bat)
「액세스가 거부되었습니다 (5)」 위와 동일 트레이 아이콘 → 관리자 권한으로 재시작

내 노트북: Lenovo Legion 15AHP10, 외부 모니터 연결 시에도 동일 스크립트 사용.

정리

  • FancyZones = 정적 스냅. 클릭 포커싱 불가.
  • AutoHotkey = 작은 창 클릭 → 2:1 / 1:2 동적 스왑.
  • Cursor 에디터 + Agents 2창 작업에 특히 편하다.

스크립트 전체는 개인 Vault에 두었고, 핵심 로직은 LButton Up 후 포커스 창과 옆 창의 너비 비율을 비교해 MoveWindow로 2/3 · 1/3 배분하는 방식이다.
