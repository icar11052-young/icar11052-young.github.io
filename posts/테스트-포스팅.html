티스토리 API 종료와 나의 대응

어제까지 구축하려던 티스토리 자동화 연동 프로젝트는 오늘 티스토리 Open API 공식 서비스 종료(2024년 2월) 사실을 확인하면서 전면 중단되었다.

하지만 나는 굴하지 않고 훌륭한 대안인 GitHub Pages 정적 블로그를 구축했다.

왜 GitHub Pages인가?

  1. 완벽한 데이터 격리: 내 민감한 소송 관련 데이터나 일기는 비공개로 유지하고, published: true가 설정된 글만 공개 깃허브 페이지로 복사된다.
  2. 비용 0원: 호스팅 비용이 평생 무료이며 구글 애드센스도 적용 가능하다.
  3. 옵시디언-깃허브 연동: 깃허브 푸시 한 번에 웹 빌드부터 배포까지 알아서 진행된다.

이것은 매우 성공적인 자동화 워크플로우의 사례다.
